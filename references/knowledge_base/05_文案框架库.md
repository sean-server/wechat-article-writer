# 文案框架库

> 「萧十一说」专属文案框架库
> 最后更新：2026-03-21
> 包含：20+经过验证的文案结构模板，结尾模块支持多风格变体

---

## 零、标准规范（所有文章必须遵守）

### 0.1 字数要求

| 类别 | 最少字数 | 目标字数 |
|------|---------|---------|
| 干货方法类 | 2000 字 | 2500-3000 字 |
| 情感共鸣类 | 1800 字 | 2200-2500 字 |
| 热点评论类 | 1500 字 | 2000-2500 字 |
| 深度分析类 | 2000 字 | 3000-3500 字 |

> 字数统计：正文内容（不含 frontmatter、不含结尾模块）

### 0.2 章节格式要求

文章**必须使用编号章节格式**，便于自动插入配图：

```
## 01 [小节标题]

内容...

## 02 [小节标题]

内容...

## 03 [小节标题]

内容...
```

- 章节数量：至少 6 个主要章节（鼓励 7-8 个，深度内容可更多）
- 每章节内容：200-400 字
- 章节标题：简洁有力，不超过 15 字

### 0.3 标准结尾结构

**每篇文章必须按顺序包含以下三个模块**：

#### 模块一：内容总结（划重点）

根据文章风格选取变体，**要点内容必须从文章真实内容中提炼**：

```
变体A — 干货爽感型（适合方法论/分析类）
---

## 结语

- ✅ [从文章核心观点提炼，20字以内，具体可操作]
- ✅ [从文章第二个核心观点提炼，20字以内，有数据或案例支撑更好]
- ✅ [从文章结论/行动建议提炼，20字以内，让人想收藏]

变体B — 情绪共鸣型（适合观点输出/故事类）
---

## 结语

说到底，这篇文章讲的就是三句话：

[金句1]：一句话概括文章最核心的洞察
[金句2]：一句话说明为什么要知道这个
[金句3]：一句话给读者一个行动理由或心理出口

变体C — 清单简洁型（适合清单/盘点类）
---

## 结语

这篇文章的精华：

1. [最核心的一点，10-15字]
2. [第二重要的发现，10-15字]
3. [最让人意外的一点，10-15字]
4. [最实用的建议，10-15字]
```

#### 模块二：互动引导

**必须与文章主题强相关**，从以下变体中选最自然的一个：

```
变体A — 好奇追问型（适合科技/观点/分析类）
你觉得呢？

你有没有遇到过类似的情况？或者，你身边有没有这样的例子？
欢迎留言聊聊，**我真的很想知道你的想法**。

变体B — 故事邀请型（适合职场/成长/情感类）
你呢？

有没有哪一点特别戳中你？
有没有你自己的故事想分享？我在留言区等你。

变体C — 轻松调侃型（适合热点/轻松/盘点类）
你呢？有没有哪一条戳中了你？

或者，你还有什么要补充的？
说出来，评论区见。

变体D — 深度思考型（适合深度分析/哲学类）
你呢？

如果AI真的做到了某某事（引用文章中提到的具体场景），你会怎么想？
这个话题值得认真聊聊，**来，说说你的看法**。
```

#### 模块三：参考来源

```
---

**参考来源**：

- [来源1标题](真实URL链接)
- [来源2标题](真实URL链接)
- [来源3标题](真实URL链接)

*本文参考资料均来自公开渠道，观点仅供参考。*
```

> 参考来源要求：**必须使用真实URL链接**，至少 2-3 个，按以下优先级选取：
>
> | 文章主题 | 推荐来源类型 | 推荐网站 |
> |---------|------------|---------|
> | 职场/打工/薪资 | 政府官网 + 招聘平台报告 | 国家统计局、智联招聘、BOSS直聘、LinkedIn |
> | 情感/心理 | 心理学研究 + 权威媒体 | 心理学报、中科院心理所、人民日报 |
> | AI/科技 | 官方发布 + 科技媒体 | 新华社、36kr、机器之心、量子位 |
> | 热点新闻/国际 | 权威媒体原文 | 新华社、央视新闻、澎湃新闻、外交部官网 |
> | 婚姻/家庭/法律 | 政府官网 + 专业媒体 | 民政部、婚姻法司法解释、人民法院报 |
> | 数据/报告引用 | 官方统计数据 | 国家统计局 data.stats.gov.cn、各部委白皮书 |
>
> ⚠️ **禁止**：使用 example.com 等占位链接；禁止虚构来源标题和链接。

## 二、框架详解

### 框架01：痛点-方案型

**最适合干货方法类内容，爆款率最高**

```
结构：
痛点引入 → 问题分析 → 解决方案 → 行动号召

适用话题：
- 职场技能提升
- 时间管理
- 沟通技巧
- 学习方法

开头模板：
"你是不是也这样......"
"扎心了，最近很多人问我......"
"说个扎心的真相：......"

结尾模板：
"所以啊，如果你......可以试试"
"希望今天的分享对你有帮助"
```

**案例**：
> 标题：开会总是抓不住重点？试试这个万能框架
> 开头：你有没有发现，有些人说话5分钟你能记住全部，有些人说了30分钟你一句都没听懂？
> 正文：问题在哪 → 原因是啥 → 解决方法（三步）
> 结尾：赶紧试试，下次开会你也能条理清晰

---

### 框架02：故事-感悟型

**最适合个人成长、情感共鸣类内容**

```
结构：
故事引入 → 过程描写 → 转折/感悟 → 总结升华

适用话题：
- 职场经历
- 人生选择
- 成长蜕变
- 踩坑教训

开头模板：
"说个真事......"
"我有个朋友......"
"之前我......"

结尾模板：
"后来我才明白......"
"说这么多，就一句话：......"
"希望你也......"
```

**案例**：
> 标题：工作10年才明白：领导喜欢的是这种人
> 开头：我换过四五份工作，遇到过各种领导。说实话，前几年一直觉得领导都是摆设。
> 正文：讲一个关键领导的的故事 → 他教会我什么 → 顿悟时刻
> 结尾：所以啊，跟对领导比找对工作更重要

---

### 框架03：悬念-揭秘型

**最适合热点评论、揭秘类内容**

```
结构：
抛出悬念 → 层层铺垫 → 反转揭秘 → 总结点评

适用话题：
- 热点事件
- 行业揭秘
- 认知颠覆
- 背后真相

开头模板：
"很多人不知道......"
"为什么......？今天说个真相"
"有个现象很奇怪......"

结尾模板：
"说到底......"
"所以啊......"
"这就是......的真相"
```

**案例**：
> 标题：为什么越努力的人，越容易焦虑？
> 开头：努力就会成功，这是我们从小听到大的道理。但你有没有发现，身边那些最努力的人，往往也是最焦虑的一群人？
> 正文：列举现象 → 分析原因（3个层次） → 揭示真相
> 结尾：努力重要，但方向和方法更重要

---

### 框架04：对比反差型

**观点输出类内容的爆款框架**

```
结构：
A现象 → B现象 → 对比分析 → 结论

适用话题：
- 职场对比
- 人生选择
- 认知差异
- 成败对比

开头模板：
"月薪3000和30000的人，差在哪里？"
"有两种人......你是哪一种？"
"为什么......和......差这么多？"

结尾模板：
"所以啊......"
"选择比努力更重要"
"关键在于......"
```

**案例**：
> 标题：为什么有的人35岁危机，有的人35岁起飞？
> 开头：同样35岁，为什么有的人开始担心被裁员，有的人却越活越值钱？
> 正文：对比两种人的行为模式 → 分析差距在哪 → 给出建议
> 结尾：差距不是一天形成的，想35岁起飞，现在就得开始

---

### 框架05：清单型

**干货汇总类内容，粉丝收藏率最高**

```
结构：
引言 → 清单1 → 清单2 → ... → 清单N → 总结

适用话题：
- 方法汇总
- 工具推荐
- 技巧盘点
- 避坑指南

开头模板：
"今天整理了......分享给你"
"想要......？你只需要这X招"
"关于......，看这一篇就够了"

结尾模板：
"以上就是......如果有用，帮我点个赞"
"还有补充的？欢迎留言"
```

**案例**：
> 标题：30岁之前，你必须明白的15个人生道理
> 开头：工作10年，换过很多坑，今天把最重要的15条经验整理出来
> 正文：1. 关于职场... 2. 关于金钱... 3. 关于成长... 共15条
> 结尾：有些道理，知道了和做到之间还隔着很多年。共勉

---

### 框架06：场景-共鸣型

**情感共鸣类内容的杀手锏**

```
结构：
场景描写 → 情绪共鸣 → 原因分析 → 出路指引

适用话题：
- 情感困惑
- 职场焦虑
- 人生迷茫
- 生活压力

开头模板：
"你有没有这种感觉......"
"不知道从什么时候开始......"
"凌晨2点，你为什么还睡不着？"

结尾模板：
"但其实啊......"
"没关系，你不是一个人"
"我们都是这样过来的"
```

**案例**：
> 标题：深夜11点，你为什么还在加班？
> 开头：不知道从什么时候开始，"下班"变成了一件奢侈的事。
> 正文：描述场景（工作群、deadline、疲惫）→ 分析为什么（效率低？活太多？）→ 出路（学会拒绝/提升效率/换工作）
> 结尾：加班不是勋章，但有时候也是无奈。一起加油吧，打工人

---

### 框架07：递进型

**深度分析类内容，打造专业IP**

```
结构：
现象 → 第一层分析 → 第二层分析 → 第三层分析 → 结论

适用话题：
- 行业趋势
- 深度分析
- 复杂问题
- 系统性思考

开头模板：
"很多人只看到......但其实......"
"要理解......，首先要......"
"今天说个很多人没想明白的事"

结尾模板：
"说到底......"
"这就是......的底层逻辑"
```

**案例**：
> 标题：为什么35岁危机越来越提前了？
> 开头：这几年有个现象，35岁危机从35岁提前到了30岁。为什么？
> 正文：现象层（招聘要求）→ 原因层（供需变化、技术冲击）→ 本质层（人力资本vs生理资本）→ 趋势层
> 结尾：危机不可怕，可怕的是没有准备。早点想明白，早点做准备

---

### 框架08：问答型

**答疑解惑类内容，互动率高**

```
结构：
问题引入 → 分点回答 → 总结建议

适用话题：
- 读者提问
- 常见困惑
- 方法答疑
- 经验分享

开头模板：
"最近很多人问我......"
"后台收到一个很有代表性的问题"
"问的最多的就是......"

结尾模板：
"还有什么问题，欢迎留言"
"如果对你有帮助，点个在看"
```

**案例**：
> 标题：工作不开心，要不要辞职？
> 开头：这是最近被问最多的问题。今天统一回答一下
> 正文：什么情况下可以考虑辞职？什么情况下不建议辞？辞职前要准备什么？
> 结尾：没有标准答案，但有思考框架。希望对你有帮助

---

### 框架09：金句开头型

**万能开头框架，几乎所有内容都适用**

```
结构：
金句 → 解读/展开 → 论述 → 结论/行动

适用话题：
- 所有类型

开头模板：
"说个扎心的真相：......"
"说到底，......就一个字"
"有句话说得对：......"
"懂的人都懂：......"

结尾模板：
"希望你也......"
"一起加油"
"共勉"
```

**案例**：
> 标题：选择比努力更重要，但选择本身就是一种努力
> 开头：选择比努力更重要，这句话我相信你听过很多遍。但今天我想说另一面：选择本身，就是一种努力。
> 正文：为什么选择是努力？怎么做更好的选择？
> 结尾：愿你每一个选择，都是深思熟虑后的结果

---

### 框架10：数据驱动型

**分析报告类内容，可信度高**

```
结构：
数据引入 → 数据解读 → 原因分析 → 建议行动

适用话题：
- 行业分析
- 趋势报告
- 数据新闻
- 研究发现

开头模板：
"有个数据很有意思......"
"最新报告显示......"
"根据......的研究......"

结尾模板：
"数据不会说谎，关键看你怎么解读"
"希望这个分析对你有参考价值"
```

**案例**：
> 标题：2026年轻人职场报告：月薪多少才不焦虑？
> 开头：最近看到一个报告，调研了10000个职场人。关于收入和焦虑，有几个数据很有意思
> 正文：数据1（收入分布）→ 数据2（焦虑指数）→ 数据3（影响因素）→ 分析
> 结尾：数字是参考，不是标准。你的焦虑，不是靠赚钱能解决的

## 六、框架使用 checklist

### 写之前

```
[ ] 确定内容类型
[ ] 选择合适框架
[ ] 列出核心观点（不超过3个）
[ ] 准备案例/故事素材
```

### 写之中

```
[ ] 开头有钩子（痛点/故事/悬念/金句）
[ ] 中间有支撑（数据/案例/理论）
[ ] 结尾有金句（让人想收藏）
[ ] 互动引导（让人想留言）
```

### 写之后

```
[ ] 通读一遍，像不像和朋友聊天
[ ] 检查有没有太说教的句子
[ ] 看看有没有让人"扎心"的地方
[ ] 标题是否吸引人点击
```

### 结尾句式

```
总结型：
"所以啊......"
"说这么多，就一句话：......"

金句型：
"愿你......"
"希望你也......"
"一起加油......"
"共勉"

互动型：
"你呢？欢迎留言......"
"你有过类似的经历吗？"
```
