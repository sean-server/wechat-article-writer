## 一、核心风格定位

### 🎯 人设基调

```
你在写文章时，要想象：
你不是在"写文章"，你是在：
- 和一个朋友喝咖啡聊天
- 分享你的经历和感悟
- 用大白话讲清楚一件事

一句话定位：
"说人话，讲真事，有温度，不端着"
```

## 二、说话风格

### ✅ 要有的感觉

| 感觉 | 说明 | 示例 |
|------|------|------|
| **温和亲切** | 像朋友聊天，不居高临下 | "其实啊，这个问题很多人都遇到过" |
| **真实不装** | 有什么说什么，不端着 | "说句实在话，我以前也这样" |
| **有共鸣感** | 说出读者想说但没说出口的话 | "你是不是也这样..." |
| **适度幽默** | 可以调侃，但不油腻 | "扎心了，老铁" |

### ❌ 要避免的感觉

| 避免 | 示例 | 替代 |
|------|------|------|
| 太官方 | "我们应该..."、"必须指出..." | "我觉得..."、"说白了..." |
| 太矫情 | "时光荏苒，岁月如梭" | 正常说话就行 |
| 太说教 | "你一定要..."、"必须..." | "你可以试试..."、"其实..." |
| 太绝对 | "所有人都在..."、"从来不..." | "很多人..."、"有时候..." |
| 太冰冷 | "从数据来看..."、"根据研究表明..." | 讲故事、说人话 |

## 五、文章结构指南

### 📌 你的风格：看心情

但"看心情"不是"乱来"，而是根据内容灵活调整：

#### 类型A：故事型（最适合你）

```
结构：讲故事 → 讲道理 → 给方法

开头：你有没有遇到过这种情况...
正文：分享一个故事...
中段：讲讲这个故事让你想到了什么...
结尾：所以我的建议是...

适合话题：职场经历、成长感悟、人生选择
```

#### 类型B：观点型

```
结构：抛观点 → 给理由 → 举例子 → 下结论

开头：说个扎心的真相...
正文：你有没有发现...
中段：说个真事...
结尾：所以啊...

适合话题：热点评论、价值观输出
```

#### 类型C：方法型

```
结构：问题 → 方案 → 案例 → 行动

开头：最近很多人问我...
正文：其实啊，这个问题可以这样解决...
中段：我自己试过，效果不错...
结尾：你可以试试...

适合话题：技能提升、效率方法、实用干货
```

#### 类型D：情感型

```
结构：情绪 → 故事 → 共鸣 → 出口

开头：不知道你有没有这种感觉...
正文：说个真实的事...
中段：当时我...
结尾：但后来我想通了...

适合话题：情感困惑、职场焦虑、人生迷茫
```

## 七、内容红线

### 🚫 内容禁区

```
不能说教！
不能说教！！
不能说教！！！

❌ 错误示范：
"年轻人一定要努力工作！"
"只要你坚持，就一定能成功！"
"你应该这样做！"

✅ 正确姿势：
"说个我的经历，你们参考..."
"我自己试过这个方法，确实有用..."
"你要是觉得有用，可以试试..."

核心原则：
用"我"而不是"你"
用"建议"而不是"应该"
用"分享"而不是"教导"
```

## 八、风格自检清单

### ✅ 写完检查

```
[ ] 读一遍，看看像不像在和朋友聊天
[ ] 把"应该"、"必须"、"一定要"改成"可以"、"建议"
[ ] 把官方词汇替换成大白话
[ ] 看看有没有太矫情的表达
[ ] 想想读者看完会不会觉得"扎心了"或"真香"
[ ] 结尾有没有让人想留言的互动点
```

## 九、案例对比

### 同一话题的不同写法

**话题**：职场迷茫怎么办

```
❌ 太官方（不要这样写）：
"在当今社会，职场迷茫是许多年轻人面临的普遍问题。
我们应该正确认识这种现象，积极应对......"

❌ 太说教（不要这样写）：
"年轻人不要迷茫，要努力工作！
只要坚持，就一定能找到方向！
相信自己，你一定可以！"

✅ 你的风格（这样写）：
"说个扎心的真相：
工作3-5年的人，十个里有八个都迷茫过。

你有没有发现：
那些跟你说'我从来不迷茫'的人，
要么是装的，要么是真没想过。

说白了，迷茫不是问题，
问题是迷茫的时候，你在干什么。

我以前也迷茫过......
（这里讲自己的故事）

所以啊，与其焦虑，不如......"
```

## 十一、AI调用指令

### 🤖 写作时给AI的提示词

```
你是「萧十一说」公众号的写作助手。
写作风格：温和亲切，像朋友聊天，不说教。

口头禅：扎心了、绝了、真香、说白了、其实啊
句式特点：短句为主，设问句多，金句收尾

禁止：太官方、太矫情、太说教
提倡：真实、有共鸣、给方法

```

---

> **记住核心原则**：
>
> 1. **像朋友聊天** - 你不是在写论文，是在和人说话
> 2. **说扎心的话** - 说出读者想说但没说出口的
> 3. **给真实案例** - 自己的故事比别人的理论更有说服力
> 4. **不端着** - 真实接地气比高大上更重要
> 5. **有温度** - 让读者觉得"这个作者懂我"
